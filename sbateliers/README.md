SbAteliers est une application web permettant aux clients de pouvoir consulter la liste des ateliers et de s'y inscrire. De plus, cela permet aux responsables des ateliers de gérer les ateliers pour les programmer, consulter le suivi des ateliers ou annuler des ateliers.

Ce projet m'a permis d’utiliser :
- PHP
- HTML

Le modèle MVC (Modèle-Vue-Controller)
